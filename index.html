We offer custom air quality measuring solutions using wireless sensor-networks. We provide both indoor and outdoor solutions, giving a real-time reading of air quality on a Cloud database. Parameters measured include temperature, humidity, particle matter and various gases such as CO2, CO, NO and Ozone.

The sensor nodes employ various communication methods, including 868 MHz long-range radio, which has a transmission range of more than 500 meters. An easy to use Cloud interface provides real-time readings and a GPS location of every node in the network. This allows to set up networks of any size, whether it be a home-based network or one that covers an entire city.
